# 26. 删除有序数组中的重复项

## 同向双指针

时间复杂度：O(n)。

```java
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int i = 0, j = 0;
        int len = nums.size();
        while (j < len) {
            if (j && nums[j] == nums[j-1]) {
                j++;
            } else {
                nums[i++] = nums[j++];
            }
        }
        return i;
    }
};
```
