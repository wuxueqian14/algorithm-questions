# 28. Find the Index of the First Occurrence in a String 

## 暴力

时间复杂度：O(nm)。

```c
class Solution {
public:
    int strStr(string haystack, string needle) {
        int hlen = haystack.size(), nlen = needle.size();
        for (int i = 0; i < hlen; i++) {
            int j = i, k = 0;
            while (k < nlen && haystack[j] == needle[k]) {
                j++;
                k++;
            }
            if (k == nlen) {
                return i;
            }
        }
        return -1;
    }
};
```
