# 27. 移除元素

## 同向双指针

时间复杂度：O(n)。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int idx = 0;
        for (int i = 0; i < nums.length; i += 1) {
            if (nums[i] != val) {
                nums[idx] = nums[i];
                idx += 1;
            }
        }
        return idx;
    }
}
```
