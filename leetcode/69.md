# 69. x的平方根

## 二分查找

染色法，左区间为平方小于等于x，又区间为平方大于x，寻找边界。

时间复杂度：O(logn)。


```cpp
class Solution {
public:
    int mySqrt(int x) {
        int l = 0, r = x;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long long) mid * mid <= x) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return r;
    }
};
```
