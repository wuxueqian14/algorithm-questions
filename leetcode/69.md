# 69. x的平方根

## 二分查找

时间复杂度：O(logn)。

左区间为平方小于等于x，右区间为平方大于x。空间为[0, x]，因为0必在左区间，所以初始空间为[1, x]。

```java
class Solution {
    public int mySqrt(int x) {
        int l = 1, r = x;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long) mid * mid <= x) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return r;
    }
}
```
