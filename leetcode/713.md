# 713. 乘积小于 K 的子数组

## 双指针 滑动窗口

[l, r]计算以r为终点的连续子数组个数，个数为[l, r] [l+1, r] ... [r, r]一共r-l+1个。

时间复杂度：O(n)。

```java
class Solution {
    public int numSubarrayProductLessThanK(int[] nums, int k) {
        int len = nums.length;
        int l = 0, r = 0;
        int sum = 1;
        int ans = 0;
        while (r < len) {
            sum *= nums[r];
            while (l <= r && sum >= k) {
                sum /= nums[l++];
            }
            ans += (r - l + 1);
            r++;
        }
        return ans;
    }
}
```
