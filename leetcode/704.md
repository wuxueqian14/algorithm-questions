# 704. 二分查找

## 红蓝染色法

l和r指针指向边界，左区间为蓝色，右区间为红色。满足条件移动指针，更新染色区域。循环结束时l指向蓝边界，r指向红边界。

```cpp
l = -1, r = N
while (l + 1 != r) {
    m = (l + r) / 2;
    if (isBlue(m)) {
        l = m;
    } else {
        r = m;
    }
    return l or r;
}
```

## 二分查找

小于等于target的为蓝区间，大于target的为红区间。

时间复杂度：O(logn)。  

```cpp
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int l = -1, r = nums.size(), mid;
        while (l + 1 != r) {
            mid = l + (r - l) / 2;
            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                l = mid;
            } else {
                r = mid;
            }
        }
        return -1;
    }
};
```
