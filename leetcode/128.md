# 128. 最长连续序列

## 哈希表

哈希表维护数和次数，从每个数向左右寻找更新最大区间。

时间复杂度：O(n)。

空间复杂度：O(n)。

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_map<int, int> m;
        for (auto num : nums) {
            m[num]++;
        }
        int ans = 0;
        for (auto &t : m) {
            if (t.second != 0) {
                int i = t.first + 1;
                int cnt = 1;
                while (m.count(i)) {
                    m[i] = 0;
                    cnt++;
                    i++;
                }
                i = t.first - 1;
                while (m.count(i)) {
                    m[i] = 0;
                    cnt++;
                    i--;
                }
                ans = max(ans, cnt);
            }
        }
        return ans;
    }
};
```
