# 128. 最长连续序列

## 哈希表

哈希表记录，中间向两边扩散。

时间复杂度：O(n)。

空间复杂度：O(n)。

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_map<int, int> m;
        for (auto x : nums) {
            m[x] = 1;
        }
        int ans = 0;
        for (int i = 0; i < nums.size(); i++) {
            int j = nums[i] - 1;
            int cnt = 1;
            while (m.count(j)) {
                m.erase(j);
                cnt++;
                j--;
            }
            j = nums[i] + 1;
            while (m.count(j)) {
                m.erase(j);
                cnt++;
                j++;
            }
            ans = max(ans, cnt);
        }
        return ans;
    }
};
```
