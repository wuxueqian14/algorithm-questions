# 416. 分割等和子集

## 01背包

计算01背包价值=sum/2

时间复杂度：O(n * target)。

空间复杂度：O(target)。

```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = 0;
        for (int x : nums) {
            sum += x;
        }
        if (sum % 2) {
            return false;
        }
        int size = nums.size();
        sum /= 2;
        int dp[sum+1];
        memset(dp, 0, sizeof(dp));
        for (int i = 0; i < size; i++) {
            for (int j = sum; j >= nums[i]; j--) {
                dp[j] = max(dp[j], dp[j-nums[i]]+nums[i]);
            }
        }
        return dp[sum] == sum;
    }
};
```
