# 383. 赎金信

## 哈希表

判断s串是否能由t串字符构成，使用哈希表维护t串字符和出现次数，然后对s串检查即可。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    bool canConstruct(string ransomNote, string magazine) {
        unordered_map<char, int> m;
        for (auto c : magazine) {
            m[c]++;
        }
        for (auto c : ransomNote) {
            if (!m.count(c)) {
                return false;
            } 
            m[c]--;
            if (m[c] == 0) {
                m.erase(c);
            }
        }
        return true;
    }
};
};
```
