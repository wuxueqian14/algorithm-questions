# 48. 旋转图像

## 规律

(i, j) -> (j, n-i-1)
(j, n-i-1) -> (n-i-1, n-j-1)
(n-i-1, n-j-1) -> (n-j-1, i)
(n-j-1, i) -> (i, j)

顺时针填会覆盖，逆时针填。

时间复杂度：O(n * n)。

```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();
        for (int i = 0; i < n/2; i++) {
            for (int j = 0; j < (n+1)/2; j++) {
                int t = matrix[i][j];
                matrix[i][j] = matrix[n-j-1][i];
                matrix[n-j-1][i] = matrix[n-i-1][n-j-1];
                matrix[n-i-1][n-j-1] = matrix[j][n-i-1];
                matrix[j][n-i-1] = t;
            }
        }
    }
};
```
