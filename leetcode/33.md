# 33. 搜索旋转排序数组

## 二分查找

每次二分可以确定左边或者右边为有序空间，根据target与有序空间的大小关系判断target剩下查找区间。

时间复杂度：O(logn)。

```cpp
class Solution {
public:
    bool isBlue(vector<int>& nums, int mid, int end, int target) {
        if (nums[mid] > nums[end]) {
            return target > nums[mid] || target <= nums[end];
        } else {
            return target > nums[mid] && target <= nums[end];
        }
    }
    int search(vector<int>& nums, int target) {
        int l = 0, end = nums.size() - 1, r = end;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] == target) {
                return mid;
            }
            if (isBlue(nums, mid, end, target)) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return -1;
    }
};
```
