# 59. 螺旋矩阵 II

模拟每一圈，4个方向，走一样的step，只处理每条边的头结点，不处理尾结点。n为奇数需要中间填数字，模拟n/2圈。

时间复杂度：O(n*n)。

## 模拟

```cpp
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<int> t(n, 0);
        vector<vector<int>> v(n, t);
        int l = 0, r = 0;
        int idx = 1;
        int s = n - 1;
        while (s > 0) {
            for (int i = 0; i < s; i++) {
                v[l][r++] = idx++;
            }
            for (int i = 0; i < s; i++) {
                v[l++][r] = idx++;
            }
            for (int i = 0; i < s; i++) {
                v[l][r--] = idx++;
            }
            for (int i = 0; i < s; i++) {
                v[l--][r] = idx++;
            }
            s -= 2;
            l++;
            r++;
        }
        if (n % 2) {
            v[n/2][n/2] = n * n;
        }
        return v;
    }
};
```
