# 852. 山脉数组的峰顶索引

左区间为山峰左边，右区间为山峰及其右边。

## 二分查找

时间复杂度：O(logn)。

```java
class Solution {
    public int peakIndexInMountainArray(int[] arr) {
        int l = 0, r = arr.length - 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (arr[mid] < arr[mid + 1]) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return l;
    }
}
```
