# 377. 组合总和 Ⅳ

## DP

类似爬楼梯，楼梯有n种选择。

```cpp
class Solution {
public:
    int combinationSum4(vector<int>& nums, int target) {
        int dp[target+1];
        memset(dp, 0, sizeof(dp));
        dp[0] = 1;
        for (int i = 1; i <= target; i++) {
            for (int j = 0; j < nums.size(); j++) {
                if (i >= nums[j] && dp[i] < INT_MAX - dp[i-nums[j]]) {
                    dp[i] += dp[i-nums[j]];
                }
            }
        }
        return dp[target];
    }
};
```
