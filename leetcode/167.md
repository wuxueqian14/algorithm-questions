# 167. 两数之和 II - 输入有序数组

## 相向双指针

时间复杂度：O(n)。

```java
class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int l = 0, r = numbers.length - 1;
        while (l < r) {
            int t = numbers[l] + numbers[r];
            if (t == target) {
                return new int[]{l + 1, r + 1};
            } else if (t > target) {
                r--;
            } else {
                l++;
            }
        }
        return new int[]{-1, -1};
    }
}
```
