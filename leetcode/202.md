# 202. 快乐数

## 哈希表

用set维护循环中的num，如果重复不是快乐数。

时间复杂度：O(logn)。

```c
class Solution {
public:
    bool isHappy(int n) {
        unordered_set<int> s;
        while (n != 1) {
            n = getNum(n);
            if (s.find(n) == s.end()) {
                s.insert(n);
            } else {
                return false;
            }
        }
        return true;
    }

    int getNum(int n) {
        int sum = 0;
        while (n != 0) {
            sum += (n%10) * (n%10);
            n /= 10;
        }
        return sum;
    }
};
```
