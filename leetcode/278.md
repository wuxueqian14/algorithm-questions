# 278. 第一个错误的版本

## 二分查找

lowerBound函数类似。

时间复杂度：O(logn)。

```c
// The API isBadVersion is defined for you.
// bool isBadVersion(int version);

class Solution {
public:
    int firstBadVersion(int n) {
        int l = 0, r = n, mid;
        while (l + 1 != r) {
            mid = l + (r - l) / 2;
            if (!isBadVersion(mid)) {
                l = mid;
            } else {
                r = mid;
            }
        }
        return r;
    }
};
```
