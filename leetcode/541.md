# 541. 反转字符串 II

## 字符串

时间复杂度：O(n)。

```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        int len = s.size();
        for (int i = 0; i < len; i += 2*k) {
            if (i + k <= len) {
                reverseHelp(s, i, i+k-1);
            } else {
                reverseHelp(s, i, len-1);
            }
        }
        return s;
    }

    void reverseHelp(string &s, int l, int r) {
        while (l < r) {
            char t = s[l];
            s[l] = s[r];
            s[r] = t;
            l++, r--;
        }
    }
};
```
