# 3. 无重复字符的最长子串

## 滑动窗口

用一个哈希表去维护字符是否重复出现，而且有个细节如果出现重复肯定是新进来的s[r]出现重复。

时间复杂度：O(n)。

空间复杂度：O(128) / O(1)。

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        map<char, int> m;
        int l = 0;
        int ans = 0;
        for (int r = 0; r < s.size(); r++) {
            m[s[r]]++;
            while (m[s[r]] > 1) {
                m[s[l]]--;
                l++;
            }
            ans = max(ans, r-l+1);
        }
        return ans;
    }
};
```
