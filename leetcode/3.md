# 3. 无重复字符的最长子串

## 滑动窗口

哈希表维护窗口，没有出现重复字符向右移动，出现重复字符缩小窗口。

时间复杂度：O(n)。

空间复杂度：O(1)。

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> m;
        int l = 0;
        int ans = 0;
        for (int r = 0; r < s.size(); r++) {
            while (m.count(s[r])) {
                m.erase(s[l]);
                l++;
            }
            m[s[r]]++;
            ans = max(ans, r-l+1);
        }
        return ans;
    }
};
```
