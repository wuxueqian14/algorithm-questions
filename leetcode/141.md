# 141. 环形链表

## 快慢指针

设置快指针每次走2步，慢指针每次走1步，如果存在环，快指针必定会追上慢指针。

时间复杂度：O(n)。

空间复杂度：O(1)。

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
        ListNode *f = head, *s = head;
        while (f && f->next) {
            f = f->next->next;
            s = s->next;
            if (f == s) {
                return true;
            }
        }
        return false;
    }
};
```
