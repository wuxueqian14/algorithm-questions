# 移动零

## 同向双指针

时间复杂度：O(n)。

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int s = 0, f = 0;
        for (; f < nums.length; f++) {
            if (nums[f] != 0) {
                nums[s++] = nums[f];
            }
        }
        while (s < f) {
            nums[s++] = 0;
        }
    }
}
```
