# 移动零

## 同向双指针

时间复杂度：O(n)。

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int l = 0;
        for (int r = 0; r < nums.size(); r++) {
            if (nums[r] != 0) {
                nums[l++] = nums[r];
            }
        }
        for (; l < nums.size(); l++) {
            nums[l] = 0;
        }
    }
};
```
