# 20. 有效的括号

## 栈

如果是左符号，左符号入栈；如果是右符号，出栈检查是否匹配。

时间复杂度：O(n)。

空间复杂度：O(n)。

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> stack;
        for (auto ch : s) {
            if (ch == '[' || ch == '(' || ch == '{') {
                stack.push(ch);
            } else {
                if (stack.empty()) {
                    return false;
                }
                char top = stack.top();
                stack.pop();
                if (ch == ']' && top != '[') {
                    return false;
                } 
                if (ch == ')' && top != '(') {
                    return false;
                }
                if (ch == '}' && top != '{') {
                    return false;
                }
            }
        }
        return stack.empty();
    }
};
```
