# 121. 买卖股票的最佳时机

## 贪心

时间复杂度：O(n)。

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int ans = 0;
        int min_price = INT_MAX;
        for (auto price : prices) {
            min_price = min(min_price, price);
            ans = max(ans, price-min_price);
        }
        return ans;
    }
};
```
