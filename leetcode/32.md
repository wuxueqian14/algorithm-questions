# 32. 最长有效括号

# DP

dp[i]表示以i结尾的最长有效长度。

在s[i]是右括号的基础上，看s[i-1]。

```cpp
class Solution {
public:
    int longestValidParentheses(string s) {
        if (s.empty()) {
            return 0;
        }
        int size = s.size();
        int dp[size+2];
        memset(dp, 0, sizeof(dp));
        int ans = 0;
        for (int i = 1; i < size; i++) {
            if (s[i] == ')') {
                if (s[i-1] == '(') {
                    dp[i+2] = dp[i] + 2;
                } else if (i-dp[i+1]-1 >= 0 && s[i-dp[i+1]-1] == '(') {
                    dp[i+2] = dp[i+1] + dp[i-dp[i+1]] + 2;
                }
            }
            ans = max(ans, dp[i+2]);
        }
        return ans;
    }
};
```
