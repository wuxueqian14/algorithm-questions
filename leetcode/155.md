# 155. 最小栈

## 栈

时间复杂度：O(1)。

使用辅助栈，每插入一个元素记录当前最小值。

```cpp
class MinStack {
private:
    stack<int> s1, s2;
public:
    MinStack() {
        s2.push(INT_MAX);
    }
    
    void push(int val) {
        if (val < s2.top()) {
            s1.push(val);
            s2.push(val);
        } else {
            s1.push(val);
            s2.push(s2.top());
        }
    }
    
    void pop() {
        s1.pop();
        s2.pop();
    }
    
    int top() {
        return s1.top();
    }
    
    int getMin() {
        return s2.top();
    }
};

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack* obj = new MinStack();
 * obj->push(val);
 * obj->pop();
 * int param_3 = obj->top();
 * int param_4 = obj->getMin();
 */
```
