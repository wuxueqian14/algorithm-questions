# 162. Find Peak Element

## 二分查找

蓝色区间为峰顶左边元素，红色区间为峰顶及其右侧元素，最后一个元素是红色区间（存在至少一个峰顶，峰顶在中间，最后一个元素是红色；如果峰顶在最后一个元素当然也是红色）。

初始区间设置为(-1, n-1)。

时间复杂度：O(logn).

```c
class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int l = -1, r = nums.size() - 1, m;
        while (l + 1 != r) {
            m = l + (r - l) / 2;
            if (nums[m] < nums[m+1]) {
                l = m;
            } else {
                r = m;
            }
        }
        return r;
    }
};
```
