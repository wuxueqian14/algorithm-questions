# 162. Find Peak Element

## 二分查找

时间复杂度：O(logn).

```c
class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int l = 0, r = nums.size() - 2, mid;
        while (l <= r) {
            mid = l + (r - l) / 2;
            if (nums[mid] > nums[mid+1]) {
                r = mid - 1;
            } else {
                l = mid + 1;
            }
        }
        return l;
    }
};
```
