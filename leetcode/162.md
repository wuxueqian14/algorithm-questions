# 162. 寻找峰值

## 二分查找

左区间为峰顶左侧，右区间为峰顶及其右侧，进行染色。因为必然存在峰顶，初始时最右侧元素在右区间中。

时间复杂度：O(logn).

```cpp
class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int l = 0, r = nums.size() - 2;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] < nums[mid+1]) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return l;
    }
};
```
