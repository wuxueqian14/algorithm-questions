# 560. 和为 K 的子数组

## 前缀和+哈希表

找`s[j] - s[i] = k`，遍历s[j]哈希保存，找s[i]。

时间复杂度：O(n)。

空间复杂度：O(n)。

```cpp
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        unordered_map<int, int> m;
        int ans = 0, s = 0;
        m[0] = 1;
        for (int x : nums) {
            s += x;
            if (m.count(s - k)) {
                ans += m[s - k];
            }
            m[s]++;
        }
        return ans;
    }
};
```
