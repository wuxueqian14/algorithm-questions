# 76. 最小覆盖子串

## 滑动窗口

用哈希表维护字符的出现和次数，不满足条件右扩，满足条件左缩。

n是s的长度，m是t的长度，c是哈希表的大小。

时间复杂度：O(n*c+m)。

空间复杂度：O(c)。

```c
class Solution {
public:
    map<char, int> m1, m2;
    bool check() {
        for (auto t : m1) {
            if (m2[t.first] < t.second) {
                return false;
            }
        }
        return true;
    }
    string minWindow(string s, string t) {
        for (auto c : t) {
            m1[c]++;
        }
        int l = 0;
        int min_len = s.size() + 1;
        int ans_l;
        for (int r = 0; r < s.size(); r++) {
            if (m1.count(s[r])) {
                m2[s[r]]++;
            }
            while (check() && l <= r) {
                if ((r-l+1) < min_len) {
                    min_len = r-l+1;
                    ans_l = l;
                }
                m2[s[l]]--;
                l++;
            }
        }
        if (min_len != s.size() + 1) {
            return s.substr(ans_l, min_len);
        } else {
            return "";
        }
    }
};
```
