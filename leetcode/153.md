# 153. Find Minimum in Rotated Sorted Array

## 二分查找

最小值及其右边区域为红色区域，最小值左边区域为蓝色区域。

初始区间为(-1, n-1)，最后一个元素为红色区域。

时间复杂度：O(logn)。

```c
class Solution {
public:
    int findMin(vector<int>& nums) {
        int l = -1, r = nums.size() - 1, m;
        while (l + 1 != r) {
            m = l + (r - l) / 2;
            if (nums[m] > nums[r]) {
                l = m;
            } else {
                r = m;
            }
        }
        return nums[r];
    }
};
```
