# 55. 跳跃游戏

## 贪心

计算最大覆盖范围。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        // 最远的位置
        int end = 0;
        for (int i = 0; i < nums.size(); i++) {
            if (i <= end) {
                end = max(end, i+nums[i]);
                if (end >= nums.size()-1) {
                    return true;
                }
            }
        }
        return false;
    }
};
```
