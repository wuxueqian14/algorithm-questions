# 55. 跳跃游戏

## 贪心

计算最大覆盖范围。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int size = nums.size();
        if (size == 1) {
            return true;
        }
        int cover = 0;
        for (int i = 0; i <= cover; i++) {
            cover = max(cover, i+nums[i]);
            if (cover >= size-1) {
                return true;
            }
        }
        return false;
    }
};
```
