# 53. 最大子数组和

## 贪心

连续和为负数直接放弃，继续加会使和变小。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int len = nums.size();
        int sum = 0;
        int ans = INT_MIN;
        for (int i = 0; i < len; i++) {
            sum += nums[i];
            ans = max(sum, ans);
            if (sum < 0) {
                sum = 0;
            }
        }
        return ans;
    }
};
```

## DP

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int size = nums.size();
        int dp[size+1];
        memset(dp, 0, sizeof(dp));
        int ans = INT_MIN;
        for (int i = 0; i < size; i++) {
            dp[i+1] = max(dp[i]+nums[i], nums[i]);
            ans = max(ans, dp[i+1]);
        }
        return ans;
    }
};
```
