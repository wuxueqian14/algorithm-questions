# 62. 不同路径

## DP

时间复杂度：O(m * n)。

空间复杂度：O(m * n)。

```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        int dp[m+1][n+1];
        memset(dp, 0, sizeof(dp));
        dp[1][1] = 1;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 && j == 0) {
                    continue;
                }
                dp[i+1][j+1] = dp[i][j+1] + dp[i+1][j];
            }
        }
        return dp[m][n];
    }
};
```

## 优化

压缩到一维数组。

时间复杂度：O(m * n)。

空间复杂度：O(n)。


```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        int dp[n+1];
        memset(dp, 0, sizeof(dp));
        dp[1] = 1;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                dp[j+1] = dp[j+1] + dp[j];
            }
        }
        return dp[n];
    }
};
```
