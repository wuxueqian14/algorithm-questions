# 209. 长度最小的子数组

## 滑动窗口

不断向右缩小左端点，由于所有数是整数，可以记录上次的结果。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int l = 0;
        int sum = 0;
        int ans = nums.size() + 1;
        for (int r = 0; r < nums.size(); r++) {
            sum += nums[r];
            while (sum >= target) {
                ans = min(r - l + 1, ans);
                sum -= nums[l];
                l++;
            }
        }
        if (ans == nums.size() + 1) {
            return 0;
        }
        return ans;
    }
};
```
