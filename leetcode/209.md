# 209. 长度最小的子数组

## 同向双指针

不断向右缩小左端点，由于所有数是整数，可以记录上次的结果。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int len = nums.size();
        int l = 0;
        int sum = 0;
        int ans = len + 1;
        for (int r = 0; r < len; r++) {
            sum += nums[r];
            while (sum >= target) {
                ans = min(ans, r-l+1);
                sum -= nums[l++];
            }
        }
        if (ans == len + 1) {
            return 0;
        }
        return ans;
    }
};
```
