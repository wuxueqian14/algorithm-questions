# 209. Minimum Size Subarray Sum

## 暴力

枚举左端点，向右扩展，双循环计算。会超时。

时间复杂度：O(n)。

## 同向双指针

不断向右缩小左端点，由于所有数是整数，可以记录上次的结果。

时间复杂度：O(n)。

```c
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int len = nums.size();
        int l = 0, r = 0, sum = 0, ans = nums.size() + 1;
        for (r = 0; r < len; r++) {
            sum += nums[r];
            while (sum >= target) {
                ans = min(ans, r - l + 1);
                sum -= nums[l++];
            }
        }
        if (ans <= len) {
            return ans;
        } else {
            return 0;
        }
    }
};
```
