# 209. 长度最小的子数组

## 滑动窗口

不断向右缩小左端点，由于所有数是整数，可以记录上次的结果。

时间复杂度：O(n)。

```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int l = 0, r = 0, len = nums.size();
        int sum = 0;
        int ans = len + 1;
        while (r < len) {
            sum += nums[r];
            while (sum >= target && l <= r) {
                ans = min(ans, r-l+1);
                sum -= nums[l];
                l++;
            }
            r++;
        }
        if (ans == len + 1) {
            return 0;
        }
        return ans;
    }
};
```
