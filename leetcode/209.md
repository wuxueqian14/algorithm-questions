# 209. 长度最小的子数组

## 同向双指针

不断向右缩小左端点，由于所有数是整数，可以记录上次的结果。

时间复杂度：O(n)。

```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int l = 0, r = 0;
        int len = nums.length;
        int sum = 0, ans = len;
        boolean f = false;
        while (r < len) {
            sum += nums[r];
            while (l <= r && sum >= target) {
                f = true;
                ans = Math.min(ans, r - l + 1);
                sum -= nums[l++];
            }
            r++;
        }
        if (!f) {
            return 0;
        }
        return ans;
    }
}
```
