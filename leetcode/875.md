# 875. 爱吃香蕉的珂珂

## 二分查找

二分区间从1到每摞香蕉最大数，区间是线性的当速度k满足，大于k的也满足。

时间复杂度：O(nlogn)。

```java
class Solution {
    public int minEatingSpeed(int[] piles, int h) {
        int l = 1, r = maxPile(piles);
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (isRed(piles, mid, h)) {
                r = mid - 1;
            } else {
                l = mid + 1;
            }
        }
        return l;
    }

    private boolean isRed(int[] piles, int mid, int h) {
        long time = 0;
        for (int i = 0; i < piles.length; i += 1) {
            time += (piles[i] + mid - 1) / mid;
        }
        return time <= h;
    }

    private int maxPile(int[] piles) {
        int ans = piles[0];
        for (int i = 1; i < piles.length; i += 1) {
            if (piles[i] > ans) {
                ans = piles[i];
            }
        }
        return ans;
    }
}
```
