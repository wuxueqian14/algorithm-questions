# 875. 爱吃香蕉的珂珂

## 二分查找

二分区间从1到每摞香蕉最大数，区间是线性的当速度k满足，大于k的也满足。

时间复杂度：O(nlogn)。

```cpp
class Solution {
public:
    bool isBlue(vector<int>& piles, int s, int h) {
        long long ans = 0;
        for (auto p : piles) {
            ans += (p + s - 1) / s;
        }
        return ans <= h;
    }
    int minEatingSpeed(vector<int>& piles, int h) {
        int r = 0;
        for (auto p : piles) {
            r = max(p, r);
        }
        int l = 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (isBlue(piles, mid, h)) {
                r = mid - 1;
            } else {
                l = mid + 1;
            }
        }
        return l;
    }
};
```
