# 367. 有效的完全平方数

## 二分查找

与69题类似。

时间复杂度：O(logn)。

```cpp
class Solution {
public:
    bool isPerfectSquare(int num) {
        int l = 1, r = num;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long long) mid * mid == num) {
                return true;
            } else if ((long long) mid * mid < num) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return false;
    }
};
```
